  11. List HTML5 Semantic Elements. Give explanations and examples for each these elements.

    Answer:

    figure - specifies an image with different formats.
    article - specifies an independent self-containing article.

    summary - specifies a header for the <details> element.
    mark - specifies highlighted text.
